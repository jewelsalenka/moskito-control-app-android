To start monitoring your app(s), specify MoSKito-Control WebApp address, so that the App could get the data.

Connect to MoSKito-Control WebApp:

  1. Tap (top panel) or drag the main panel to the left.
  2. In Server address field, type in MoSKito-Control WebApp address.
  3. In Port field, type in MoSKito-Control WebApp port address.
  4. Choose between http/https protocols by sliding the http switch.
  5. In case your WebApp requires Login and Password:
    • Slide the Autorization switch to the right (to turn it green).
    • Type in your Login and Password in the corresponding fields.
  6. Tap Save.
